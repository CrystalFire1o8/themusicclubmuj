# 🎵 The Music Club MUJ

Official website of **The Music Club, Manipal University Jaipur (MUJ)**.

This website serves as the official digital platform for The Music Club, providing information about the club, its events, performances, achievements, and an exclusive Core Portal for committee members to manage club operations efficiently.

## 🌐 Live Website

https://www.tmcmuj.studio

---

## ✨ Features

- 🎶 Modern and responsive user interface
- 📅 Dynamic Core Event Calendar
- 🔐 Secure Firebase Authentication
- 👥 Role-based Core Member Portal
- 🔄 Shared real-time calendar across all core members
- 📝 Event and task management
- ⏰ 12-hour time selection interface
- 📱 Fully responsive across desktop and mobile devices
- 🎨 Custom branding and animations
- 🔍 SEO optimized with sitemap, robots.txt, canonical URLs and Open Graph metadata
- 🌍 Custom domain with HTTPS (SSL secured)

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/<your-username>/<repo-name>.git
```

Navigate into the project

```bash
cd <repo-name>
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

## 🔒 Core Portal

The Core Portal is exclusively accessible to authorized committee members of The Music Club.

Features include:

- Secure login
- Password reset on first login
- Shared event calendar
- Task scheduling
- Real-time updates
- Committee collaboration

---

## 📸 About The Music Club

The Music Club (TMC) is the official music club of **Manipal University Jaipur**, dedicated to promoting musical talent through performances, competitions, workshops, collaborations, and university events.

The website acts as the club's official online presence, showcasing its journey, members, and activities while streamlining internal management through the Core Portal.
